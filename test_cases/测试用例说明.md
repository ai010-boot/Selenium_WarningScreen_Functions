- test_example_new_module.py - 新模块测试数据配置示例

## 切换数据源

如果想使用其他数据格式，只需修改 test_login_csv_driven.py 中的一行：

```python
# 当前使用CSV
@pytest.mark.parametrize("test_case", get_test_data('login', 'csv'))

# 改为JSON
@pytest.mark.parametrize("test_case", get_test_data('login', 'json'))

# 改为Excel
@pytest.mark.parametrize("test_case", get_test_data('login', 'xlsx'))

# 自动检测
@pytest.mark.parametrize("test_case", get_test_data('login'))
```

## 添加新的测试模块

1. 在 test_cases/ 目录下创建新的测试文件
2. 参考 examples/ 中的示例编写测试
3. 如需数据驱动，使用 get_test_data('模块名')

更多示例请查看 examples/ 文件夹！
